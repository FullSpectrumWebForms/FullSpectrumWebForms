RichTextBox: HtmlControlBase

Properties

Property Type Description Get/Set
Text string Get or set the text rich text box. This is unformatted and will be set directly. If the content is formatted, the text will contain the first line of the content, as unformatted { get; set; }
Contents string Get or set the content of the rich text box, this support formatting. { get; set; }

Events

Name Parameters Description
OnTextChanged (RichTextBox control, string previousText, string newText) Called when the text changed
OnContentsChanged (RichTextBox control, string previousText, string newText) Called when the content changed
