Button: HtmlControlBase

Properties

Property Type Description Get/Set
Text string Get or set the text inside of the button { get; set; }
State State Enable or disable a button. Will added the "disabled" css class. If the button is disabled, the OnButtonClicked event won't be called even if the css class is removed from the client manually by a user { get; set; }

Events

Name Parameters Description
OnButtonClicked (HtmlControlBase control) Called when the user left click on the button. Only called if the button is not disabled ( see State )
