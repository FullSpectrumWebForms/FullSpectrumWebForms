TemplateContainer: HtmlControlBase

Properties

Property Type Description Get/Set
TemplatePath string Path to the html file. This file will be requested from the client, and inlined inside the current control. If the server return 404, the TemplatePathAlternative path will be used. The html file is cached client-side so it is safe to use as many template as you want ( Ex. to generate each item of a list view ). { get; set; }
TemplatePathAlternative string Alternate path used when TemplatePath is not found. Most common use of this is to request a mobile version in the TemplatePath, and request the desktop version here. This way, if the template doesn't have a mobile version it will automaticly fall back on the desktop version. You can then add a mobile version later without changing your code { get; set; }

Methods

Method Return type Parameters Description
GetTemplateControl<T> void (string selector) where T: HtmlControlBase Create a new control of type T and link it to an html element inside the loaded template. The element is found using the provided selector ( this.element.find(selector) ).
