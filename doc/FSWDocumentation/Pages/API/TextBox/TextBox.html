TextBox: HtmlControlBase

Properties

Property Type Description Get/Set
Text string Text inside the TextBox. Updated when the user leave the control or when the enter key is pressed. You can also use InstantFeeback to update the text while the user is typing { get; set; }
Placeholder string Placeholder to put inside the textbox when there is no text in it { get; set; }
InstantFeedback TimeSpan When set, this will update the Text property serverside when the user stop typing for InstantFeedback amout of time { get; set; }
Disabled bool Used to disable the TextBox, preventing the user from changing it. This will add the "disabled" attribute to the html element { get; set; }
ReadOnly bool Used to disable the TextBox, preventing the user from changing it. This will add the "readonly" attribute to the html element { get; set; }

Events

Name Parameters Description
OnTextChanged (TextBox control, string previousText, string newText) Called when the text changed
OnEnterPressed (TextBox control, string previousText, string newText) Called when the user press enter while inside the textbox. It doesn't mean the value has changed, but it may have
