HtmlControlBase: ControlBase

Properties

Property Type Description Get/Set
CustomSelector string Must be set before the control is initialized. If set, the client will search for the control using the provided selector.
It is usefull to bind controls to existing html code 		{ get; set; }
InnerText string Set the InnerText of the control { get; set; }
CssProperties IDictionary<string, string> Css properties of the control. These will be set inside "style=" in the html code { get; set; }
Attributes IDictionary<string, string> List of attribues of the control. These will be set directly on the html element. By example, this mySpanControl.Attributes["data-test"] = "test"; will give: <span data-test="test"></span> { get; set; }
Classes IList<string> List of the css classes to put inside "class=" in the html code. Note that if a javascript library modify the classes of an element, the server will not be informed. Modifying this list server side will removed any classes the server is unware of, and add any classes that were removed client-side. { get; set; }
PreventClickEventsPropagation bool When listening to click events, the jQuery "stopPropagation" function is called client side after sending the event to the server. To prevent this, turn PreventClickEventsPropagation to true { get; set; }
HtmlDefaultTag string if set before the control is sent to the client, this will determine the html tag used. By example, set "a" to get a <a> { get; set; }
Width string Simple accessor to update CssProperties["width"] { get; set; }
Float FloatDirection Simple accessor to update CssProperties["float"] { get; set; }
Height string Simple accessor to update CssProperties["height"] { get; set; }
PaddingTop string Simple accessor to update CssProperties["padding-top"] { get; set; }
PaddingBottom string Simple accessor to update CssProperties["padding-bottom"] { get; set; }
PaddingRight string Simple accessor to update CssProperties["padding-right"] { get; set; }
PaddingLeft string Simple accessor to update CssProperties["padding-left"] { get; set; }
BackgroundColor System.Drawing.Color Simple accessor to update CssProperties["background-color"] { get; set; }
Color System.Drawing.Color Simple accessor to update CssProperties["color"] { get; set; }
Visible VisibleState Simple accessor to update CssProperties["display"] { get; set; }
InternalStyles IDictionary<string, Dictionary<string, string>> Will generate a <style></style> element inside the control with the listed classes.
The key of the IDictionary is used for the name of the css class, then the Dictionary inside is used to specify each css attribute and value 		{ get; set; }
PopupTitle string When used with FSW.Semantic, a popup will appear over the control when the user move the mouse over it. This property control the title of the popup. Do not forget to set PopupContent { get; set; }
PopupContent string When used with FSW.Semantic, a popup will appear over the control when the user move the mouse over it. This property control the content of the popup. Do not forget to set PopupTitle { get; set; }
RightClickMenu RightClickMenuOptions This property allows you to generate a custom right click menu. The menu will appear when the user right click your control. See RightClickMenuOptions for more informations { get; set; }
